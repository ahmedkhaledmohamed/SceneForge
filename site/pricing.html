Pricing

Pay per generation,
not per subscription.

SceneForge has no subscription fee. You pay only for the AI generations you use — images and video clips — through the API providers you choose. Two options: hosted APIs (Together AI) for zero setup, or self-hosted GPUs (RunPod) for lower per-unit cost.

Open source models can be self-hosted on RunPod for lower per-unit cost. Closed models are only available through Together AI's hosted API — higher quality for multi-reference composition but not self-hostable. Multi-ref models (flux-2-pro, nano-banana-pro) are key for character + reference image composition.

What you get

free

SceneForge software

Open source. No license fee. Deploy on Railway ($5/mo) or self-host.

pay

AI generation only

You pay the API provider (Together AI or RunPod) directly for each image and clip. No markup. Costs tracked per artifact in the Studio.

own

Your data, your keys

API keys stored in your profile. Generated media on your volume. No vendor lock-in — switch providers anytime.
