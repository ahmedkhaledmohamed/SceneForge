Getting started

First video in
ten minutes.

SceneForge runs as a web app. Deploy on Railway or run locally. You'll need a Together AI key for images and an OpenRouter key for cheap Seedance video clips. See pricing for a full cost breakdown.

1Deploy

Connect the repo to Railway, add a volume at /data, set TOGETHER_API_KEY and SCENEFORGE_PASSWORD in variables. Done.

# Or run locally:
$ git clone https://github.com/ahmedkhaledmohamed/SceneForge.git && cd SceneForge
$ pip install -e . && brew install ffmpeg
$ sceneforge studio

2Create a profile

A profile is your brand — style defaults and recurring characters with reference images. Go to Settings to enter your API key. Characters are global: update the refs once, every project uses them.

3How reference images work

Each scene owns its reference images. Profile characters are shared across all scenes.

Profile character (recurring subject) → sent to every scene using that character
       ↓
Scene refs (product photos, style)    → owned by the scene, dropped onto the card
       ↓
Project refs (shared background)      → sent to ALL scenes (optional)
       ↓
AI generation = character + scene refs + project refs + scene text → image

4Create a project + scenes

One project per post or video. Add scenes — each one describes a visual moment. Drop reference images directly onto the scene card when creating it, or add them after.

5Generate images

Each scene has its own "Generate images" button with model picker and cost estimate. Multi-reference composition uses character refs + scene refs to produce the image. Multiple options per scene — click to view full size, then select the best.

6Refine

Click "refine" on a scene to edit the prompt, pose, or model. Regenerating creates a new lane — you see the full history of attempts. Use generated images from one scene as references in another for iterative refinement.

7Create clips

Switch to the Clips tab. Click "+ clip" → pick a start image from any scene, optionally an end image (the AI interpolates between them). Set duration (3-10s), model, and motion prompt. Use seedance-1.5-pro via OpenRouter for the best value ($0.12/clip). Generate, download, import into your editor.

8Try for free

The fake models run the entire pipeline for $0 — real files, zero API cost. Good for testing the workflow before spending.

9Optional: self-hosted GPU

runpod-wan-i2v runs Wan2.2-TI2V-5B on a rented RTX 4090 — ~$0.10/clip vs $0.18 hosted. See runpod-worker/README.md for the setup runbook.
